Kebijakan Privasi

Aplikasi SiKesal 2 (Sistem Informasi Keluhan Masyarakat Online Kota Jambi) Aplikasi pengaduan secara online ini akan mempermudah masyarakat untuk melakukan pengaduan seperti jalan rusak, banjir, kebakaran dan lain sebagainya masyarakat bisa langsung melapor kepada aplikasi Sikesal dan akan diteruskan ke dinas terkait di Kota Jambi dan dicari penyebab serta solusinya.

Fitur Aplikasi

- Notification Laporan secara Realtime Firebase - Komentar pengguna - lebih dari 80 kategori masalah pelaporan - dapat mengirim laporan melalui sosial media , mediacetak, SMS yang akan di filter oleh Admin - ResponTime laporan pengguna

Izin Aplikasi

ACCESS_NETWORK_STATE ,INTERNET, ACCESS_COARSE_LOCATION, ACCESS_FINE_LOCATION, READ_EXTERNAL_STORAGE, CAMERA, WRITE_EXTERNAL_STORAGE aplikasi ini membutuhkan akses internet untuk mengirim laporan, dan izin pengguna untuk camera untuk mengambil gambar kejadian serta izin file gallery pengguna mengupload gambar dari gallery juga mengambil izin GPS location dari pengguna untuk mengambil titik koordinat lokasi terjadi masalah untuk mengirim laporan , dan kami juga akan melindungi data data pengguna dengan aman, aplikasi ini berguna untuk tujuan public pemerintahan tidak berbahaya dan mengandung malware, dan menjamin keamanan data pengguna
